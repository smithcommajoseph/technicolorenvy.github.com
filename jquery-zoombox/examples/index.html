1. Basic Usage

$('#example-1').zoombox();

Example 1

2. Adding content Pre zoombox opening

$('#example-2').zoombox({
    preOpen: function(e){
      $($(e.currentTarget).data('zbxTarget')).append('<p>Some Content here</p>');
    }
});

Example 2

3. Manipulating content Post zoombox opening

$('#example-3').zoombox({
    preOpen: function(e){
      $($(e.currentTarget).data('zbxTarget')).append('<p style="display:none;">I was hidden but now i\'m visible</p>');
    },
    onOpened: function(e){
      $($(e.currentTarget).data('zbxTarget')).find('p').fadeIn();
    }
});

Example 3

4. Pre zoombox close Fn

$('#example-4').zoombox({
    preOpen: function(e){
$($(e.currentTarget).data('zbxTarget')).append('<p style="display:none;">I was lost but now i\'m found, and then i\'ll be lost again</p>');
    },
    onOpened: function(e){
      $($(e.currentTarget).data('zbxTarget')).find('p').fadeIn();
    },
    preClose: function(e){
      $($(e.currentTarget).data('zbxTarget')).find('p').remove();
    }
});

Example 5

6. Customizing zoombox size

$('#example-6').zoombox({
    targetWidth: '100',
    targetHeight: '400'
});

Note: 'targetWidth' and 'targetHeight' are in pixels.

Example 6

7. Customizing zoombox position

$('#example-7').zoombox({
    targetPosX: '300',
    targetPosY: '0',
    containerParent: '#ex-7-offset-parent'
});

Note: 'targetPosX' and 'targetPosY' are in pixels relative to the container's parent.

Note: 'containerParent' will default to $(body), here we have wrapped this example in a <div> to act as our offset parent.

Example 7

8. Growing from the mouse click

$('#example-8').zoombox({
    growFromMouse: true
});

Example 8 - click (and close) multiple times on this link to see where the 'grow' points originate.

9. Customizing zoombox position using tag attributes

$('#ex-9-offset-parent a').zoombox({
    growTagAttr: 'rel',
    containerParent: '#ex-9-offset-parent'
});

Note: the tag attribute passed by 'growTagAttr' should be in 'xCoordinate, yCoordinate' format.
