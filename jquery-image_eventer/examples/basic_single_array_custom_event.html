Single array w/ custom event

This example is functionally the same as the last example, however, as you can see below you can pass your own event name as the second argument to $.image_eventer. In this example we're using 'shama_lama_ding_dong' (passed as the var cEvent).



This allows you to _potentially_ bind to something prettier than 'image_eventer::imagesLoadComplete'.

If you see K$sha, Justin Bieber, and Taylor Swift, then all went as planned.

var $body = $('body'),
    kesha = 'imgs/dean_boyers_favorites/kesha.jpg',
    bieber = 'imgs/dean_boyers_favorites/justin-bieber.jpg',
    swift = 'imgs/dean_boyers_favorites/taylor-swift.jpg',
    $kesha = $('#crouching-tiger-hidden-kesha'),
    $bieber = $('#bieberriffic'),
    $swift = $('#she-tailors-swiftly'),
    cEvent = 'shama_lama_ding_dong';

$body
  .bind(cEvent, function(e){
    $kesha.html('<img width=300 src="'+kesha+'" alt="Go Nashville!" />');
    $bieber.html('<img width=300 src="'+bieber+'" alt="Go American Idol!" />');
    $swift.html('<img width=300 src="'+swift+'" alt="Go, um, Taylor!" />');
  })
  .image_eventer([ bieber, kesha, swift ], cEvent);
