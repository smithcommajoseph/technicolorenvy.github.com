Image Eventer Examples

1. Single array

This will walk you through a very basic image eventer example. With images passed as an array.

$.image_eventer(['img1.jpg', 'img2.jpg', 'img3.jpg']);

Basic single array

2. Passing a jQuery object

With images passed by their parent.

$.image_eventer($('#Some_Id_that_exists_in_the_DOM'));

Basic jQuery parent

3. Single array w/ custom event

Images passed as an array and a custom event.

$.image_eventer(['img1.jpg', 'img2.jpg', 'img3.jpg'], 'name_of_event');

Basic single array w/ custom event

4. jQuery object parent w/ custom event

Images passed via a jQuery parent and a custom event.

$.image_eventer($('#Some_Id_that_exists_in_the_DOM'), 'name_of_event');

Basic jQ parent w/ custom event

5. Multiple arrays

Multiple collections of images. Each set will load and fire the default $.image_eventer() loaded event.

$.image_eventer({
  collections: [
    ['img1.jpg', 'img2.jpg', 'img3.jpg'],
    ['img4.jpg', 'img5.jpg', 'img6.jpg']
  ]
});

Basic mult array

6. Mutliple jQ parents

Multiple collections of images. Each set will load and fire the default $.image_eventer() loaded event.

$.image_eventer({
  collections: [
    $('#Some_Id_that_exists_in_the_DOM'),
    $('#Some_Id_that_exists_in_the_DOM')
  ]
});

Muliple jQ parents

Multiple collections w/ custom events

Multiple collections of images. Each set will load and fire it's own event custom event.

$.image_eventer({
  collections: [
    {files: ['img1.jpg', 'img2.jpg', 'img3.jpg'], completedEvent: 'name_of_event1'},
    {files: $('#Id_in_the_DOM'), completedEvent: 'name_of_event2'}
  ]
});

Multiple collections w/ custom events
