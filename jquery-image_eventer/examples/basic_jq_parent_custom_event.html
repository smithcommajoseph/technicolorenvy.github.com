jQuery object parent w/ custom event

As we've seen, you target images by passing a jQuery parent. This is the same as previous examples, but with a custom event.



When using this method All node items that are imgs will be selected for 'preloading'.

var $body = $('body'),
    $ul = $('ul');

$body
  .bind('funkyPonies', function(e){
    $ul.fadeIn().removeClass('hidden');
  })
  .image_eventer($ul, 'funkyPonies');
