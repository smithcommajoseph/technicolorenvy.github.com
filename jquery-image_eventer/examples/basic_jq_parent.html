Passing a jQuery object

You can also target images you want to load by passing a jQuery object that contains img tags.



When using this method All node items that are imgs will be selected for 'preloading'. As you'll notice, this method allows the same amount of control as passing an array of urls to your images, but allows you to write much less code.

var $body = $('body'),
    $ul = $('ul');

$body
  .bind('image_eventer::imagesLoadComplete', function(e){
    $ul.fadeIn().removeClass('hidden');
  })
  .image_eventer($ul);
